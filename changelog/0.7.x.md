# Changelog — 0.7.x Series

## v0.7.0

Kicks off the 0.7.x series with the first model-explainability feature: SHAP-based global feature-importance summaries surfaced alongside the existing impurity/coefficient-based Feature Importance view in the Experiments page. Backend gains a new optional `shap` dependency (skyulf-core exposes it as an opt-in extra) and a lightweight, training-time explainability computation that never blocks or fails a training job if SHAP can't run for a given model/dataset combination.
